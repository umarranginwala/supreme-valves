ASME B16.5 Class 150 Flange Dimensions

This table provides essential dimensional data for ASME B16.5 Class 150 flanges, including Weld Neck, Slip-On, and Blind types. Adhering to these standards ensures interchangeability and safe operation in various piping systems. This data is crucial for engineers and designers in selecting appropriate flanges for low-pressure applications.
