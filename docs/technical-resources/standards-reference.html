Standards & Reference

A comprehensive guide to international and national standards governing the design, manufacturing, testing, and application of industrial valves.

ASME B16.5 Overview

ASME B16.5 covers pressure-temperature ratings, materials, dimensions, tolerances, marking, testing, and methods of designating openings for pipe flanges and flanged fittings. It includes flanges with rating class designations 150, 300, 400, 600, 900, 1500, and 2500 in sizes NPS 1/2 through NPS 24.

View ASME B16.5

ANSI Flange Specification Summary

Historically, ANSI B16.5 was the standard for pipe flanges. While ASME B16.5 is the current designation, the term ANSI flange is still widely used and refers to the same specifications for dimensions and pressure ratings.

API Valve Standards

The American Petroleum Institute (API) sets standards for the oil and natural gas industry, including critical specifications for valves.

  • API 600: Steel Gate Valves—Flanged and Butt-welding Ends
  • API 602: Compact Carbon Steel Gate Valves—Flanged, Threaded, Welding, and Extended-Body Ends
  • API 603: Corrosion-Resistant, Bolted Bonnet Gate Valves—Flanged and Butt-Welding Ends
  • API 607: Fire Test for Quarter-Turn Valves and Other Soft-Seated Valves
  • API 609: Butterfly Valves: Double Flanged, Lug- and Wafer-Type

ISO Valve Standards Index

The International Organization for Standardization (ISO) provides global standards across various industries, including general requirements for industrial valves.

  • ISO 5208: Industrial valves — Pressure testing of metallic valves
  • ISO 5210: Industrial valves — Multi-turn valve actuator attachments
  • ISO 5752: Metal valves for use in flanged pipe systems — Face-to-face and centre-to-face dimensions

IBR (Indian Boiler Regulation) Overview for Valves

The Indian Boiler Regulations (IBR) are a set of rules and regulations that govern the design, manufacture, installation, and operation of boilers and associated components, including valves, in India. Compliance with IBR is mandatory for all valves used in steam and hot water applications above certain pressure and temperature limits.
